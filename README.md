# 米国株ひとこと投資メモ

「米国株投資ひとことメモ」は、米国の株式市場での投資を気軽に管理することができるツールです。\
ひとことで投資の意図やアイデアを記録できるので米国株投資をするにあたり手軽に利用できるのが魅力です。\
また米国株投資ひとことメモを使えば日々の投資をよりスムーズに管理できるようになります。
